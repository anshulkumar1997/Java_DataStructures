Singly Linked List
Bob and Alice are friends and interested in computer programming. One day Alice challenges Bob to create a singly linked list of N integers and find out the list is a palindrome or not. Return 1 or 0 denoting if it is  a palindrome or not, respectively.  Help Bob to win the challenge.

Note: Create the main() inside the class 'ListDriver'

Input format:
The first line of input contains an integer N denoting the size of the linked list. In the next line are N space separated values of the nodes of the linked list.

Output format:
For each test case output will be 1 if the linked list is a palindrome else 0.

Sample Input 1:
4
1 2  2 1

Sample Output1:

1

Sample Input 2:
9
3 4 2 5 6 2 7 1 9

Sample Output 2:

0
